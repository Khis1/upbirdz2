## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

To create a production build:

```bash
npm run build
```

The optimized build will be in the `build` folder.

## How to Play

### Desktop Controls
- **Arrow Keys** or **WASD** to move the bird
- **Spacebar** to pause/resume the game

### Mobile Controls
- **Swipe** on the game board in any direction to move
- **Direction buttons** below the game board
- Tap **Pause/Resume** button when available

### Game Rules
1. Control your bird to collect the purple pills ğŸ’Š
2. Each pill increases your score and makes the bird longer
3. Avoid hitting the walls or your own body
4. The game speeds up as you score more points
5. Try to achieve the highest score possible!
